# EYT Cinema Booking System

## Overview

EYT Cinema Booking System is a web application built using Spring Boot and Thymeleaf that allows users to browse movies, choose seats, book tickets, and manage their bookings online.

The system includes:

* User authentication
* Movie browsing
* Show scheduling
* Seat booking
* Ticket generation
* Booking history
* Dynamic seat availability
